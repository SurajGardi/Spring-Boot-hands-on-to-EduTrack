1. What is Lombok?
------------------

Lombok is a Java library that uses annotations to generate commonly used
code such as:
✔ getters
✔ setters
✔ constructors
✔ toString()
✔ equals() and hashCode()

Content Confirmed from Lombok PDF (Page 1 & 2): :contentReference[oaicite:0]{index=0}
- Lombok reduces boilerplate code
- Helps keep Spring Boot projects clean
- Uses annotation processing
- Supports builder pattern

----------------------------------------------------------

2. Why Lombok in Spring Boot?
-----------------------------

Spring Boot projects contain multiple:
✔ Entities
✔ DTOs
✔ Models
✔ Request/Response classes

Manually writing getters/setters is repetitive.

Lombok:
✔ reduces clutter
✔ improves readability
✔ speeds up development

3. Key Lombok Annotations Used (From Notes PDF)
-----------------------------------------------

Referenced from PDF (page 2): :contentReference[oaicite:1]{index=1}

Annotation       Purpose
---------------------------------------
@Getter          Generates getters
@Setter          Generates setters
@ToString        Generates toString()
@NoArgsConstructor No-arg constructor
@AllArgsConstructor All fields constructor
@Data            Combines Getter/Setter/ToString/Equals/HashCode
@Builder         Creates builder pattern
